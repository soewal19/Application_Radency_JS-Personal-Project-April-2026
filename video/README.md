# EventHub Demo Video

## File: `eventhub_demo.webp` (animated WebP)

This video demonstrates all major features of the EventHub application:

1. **Landing Page** — Application overview and main UI
2. **Registration** — Creating a new account  
3. **Login** — Signing in with test credentials
4. **Events List** — Browsing, searching, filtering events
5. **Event Details** — Full event info with participants
6. **Create Event** — Creating a new "Tech Meetup 2026" event
7. **My Events (Calendar)** — Calendar view of user's events
8. **Profile** — User profile and statistics
9. **Logout** — Signing out of the application
