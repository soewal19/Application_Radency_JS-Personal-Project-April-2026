## Level 4: Code (Key Patterns)

### Zustand Store Pattern
```typescript
// Single Responsibility: one store — one domain
const useEventStore = create<EventState>()((set, get) => ({
  events: [],
  fetchEvents: async (params) => { /* ... */ },
  setupSocketListeners: () => { /* ... */ },
}));
```

### NestJS Module Pattern
```typescript
// Open/Closed: modules extend via DI
@Module({
  imports: [TypeOrmModule.forFeature([Event])],
  controllers: [EventsController],
  providers: [EventsService, EventsGateway],
})
export class EventsModule {}
```

### Security Flow
```
[Client] → [Whitelist Check] → [CORS] → [JWT Guard] → [Controller] → [Service] → [DB]
```
